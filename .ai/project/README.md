# Project Specifics

Информация **именно про этот проект**, в отличие от:
- `.ai/rules/` — **как** писать код (универсально для Laravel)
- `.ai/adr/` — **почему** так решили (исторические записи)
- `.ai/memory.md` — оперативное между сессиями

Этот раздел — **что** в проекте сейчас. Живой документ, пополняется при каждом значимом изменении (см. `.ai/rules/docs.md`).

## Файлы

| Файл | Содержание |
|---|---|
| `overview.md` | 1 страница: что за проект, кто пользователи, ключевые модули |
| `glossary.md` | словарь домена (1 строка на термин) |
| `data-model.md` | ключевые таблицы, нетривиальные связи, инварианты |
| `integrations.md` | внешние API: лимиты, особенности, retry-стратегии |
| `operations.md` | окружения, очереди, cron, кэш-инвалидация |
| `gotchas.md` | известные подводные камни, которых не понять из кода |
| `runbook.md` | что делать в инцидентах |
| `domain/{module}.md` | бизнес-правила по конкретным модулям (создавать по мере необходимости) |
