# Glossary — словарь домена

> Одна строка на термин. Если описание не помещается — это уже бизнес-правило, выноси в `domain/{module}.md`.

## Формат

```
**Term** — короткое определение (1 строка). [→ domain/X.md] (опц.)
```

## Термины

*(Заполни по мере того, как термины появляются в коде и обсуждениях.)*

Пример:

- **Order** — заказ клиента, проходит статусы: pending → paid → shipped → delivered. → `domain/orders.md`
- **SKU** — уникальный код товара в нашей системе, формат `{cat}-{slug}-{NNNN}`.
- **Bonus** — бонусные баллы клиента, 1 балл = 1 копейка, не выводимы в деньги.

## Аббревиатуры

- **DTO** — Data Transfer Object
- **VO** — Value Object
- **MR / PR** — Merge Request / Pull Request
- **N+1** — типовая ошибка query-производительности
